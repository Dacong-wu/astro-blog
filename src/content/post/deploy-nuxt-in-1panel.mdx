## Nuxt 主页

1. `node .output/server/index.mjs`
2. 然后 nginx 反向代理到 http://127.0.0.1:3000/

## Console 控制页

这是基于 Vue 的后台管理页面，在 1Panel 中需要手动改配置文件

```nginx
location /console {
    alias /www/sites/ll1025.cn/index/dist; # 替换为你 Vite 项目构建后的 dist 目录路径
    try_files $uri $uri/ /console/index.html; # 为了支持 HTML5 历史模式路由，需要重定向到 index.html
}
```
